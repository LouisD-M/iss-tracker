# ISS Live Tracker

Mini application React permettant de suivre la position de la Station Spatiale Internationale en temps réel sur un globe 3D.

## Aperçu

L'application affiche la position actuelle de l'ISS à partir d'une API publique, puis la représente sur un globe interactif avec ses coordonnées principales.

## Fonctionnalités

- Globe 3D interactif
- Position de l'ISS en temps réel
- Mise à jour automatique toutes les 5 secondes
- Affichage de la latitude, longitude, altitude et vitesse
- Marqueur visuel personnalisé pour représenter l'ISS
- Interface simple et responsive

## Stack technique

- React
- TypeScript
- Vite
- react-globe.gl
- API publique Where The ISS At
